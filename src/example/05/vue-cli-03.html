npm run build 命令

首先我们来提一下，如何把写好的Vue网页放到服务器上，那我就在这里讲解一下，主要的命令就是要用到npm run build 命令。 我们在命令行中输入npm run build命令后，vue-cli会自动进行项目发布打包。你在package.json文件的scripts字段中可以看出， 你执行的npm run build命令就相对执行的 node build/build.js 。

package.json的scripts 字段：

在执行完npm run build命令后，在你的项目根目录生成了dist文件夹，这个文件夹里边就是我们要传到服务器上的文件。 dist文件夹下目录包括：

  • index.html主页文件:因为我们开发的是单页web应用，所以说一般只有一个html文件。
  • static 静态资源文件夹：里边js、CSS和一些图片。
